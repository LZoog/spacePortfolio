🏠House of Hues (Rails)

(Heroku takes 10-15 seconds to load)

Features: user authentication, view all "looks" or search by color & room, passes color & room ID through the URL, ability to submit and delete looks, redirection if user tries to create a look or go to rooms without correct parameters, matching theme on color selection, "boost" capabilities for each look, top 3 looks and 3 newest looks on home page, clicking an email (in header or another user's) brings you to a user profile page displaying all uploaded looks by that user

In code boot camp, the class was divided up into small groups to create a Ruby on Rails project required to work with user authentication and to allow users to submit something to a database. I essentially became "project leader" during this assignment by breaking up our to-do list onto small steps and assigning them via Trello, and I also did all of the front-end work and more than half of the back-end work.

In 5 days, my team and I created House of Hues: an interior design inspiration web application (hey, that almost rhymes).

Note: This project certainly isn't complete by any means. There are a few features we didn't get to. :-)
Note2: Feel free to make an account with your real or fake e-mail, they aren't used for anything.

Biggest challenge: overcoming communication blocks with the team. Things ran much more smoothly with Trello.
