Features: user authentication, view all "looks" or search by color & room, passes color & room ID through the URL, ability to submit and delete looks, redirection if user tries to create a look or go to rooms without correct parameters, matching theme on color selection, "boost" capabilities for each look, top 3 looks and 3 newest looks on home page, clicking an email (in header or another user's) brings you to a user profile page displaying all uploaded looks by that user
